# Marketing RCA — Root Causes & Hypotheses (MECE)

Dokumen ini merangkum akar masalah (RCA) dan hipotesis untuk peningkatan performa kampanye. Struktur dibuat **MECE** (Mutually Exclusive, Collectively Exhaustive).

---

## 🌳 RCA Tree (Mermaid — Mindmap)

```mermaid
mindmap
  root((Campaign Underperformance))
    A(Targeting)
      H1(Mismatch targeting  \n• Age 45–54 / Income 50–74K  \n• Higher Incremental Rev/HH than avg)
    B(Channel & Creative)
      H2(Mailer + In-store display > single channel  \n• Higher Lift% & AOV)
    C(Timing / Seasonality)
      H3(Week 40–52 yields highest Lift%)
    D(Product / Department Fit)
      H4(Grocery more responsive than Drug GM  \n• Higher Lift% & ROI)
      H5(Products with coupon_match_disc > 0  \n• Higher AOV)
    E(Discount Economics)
      H6(Too shallow/deep discounts depress ROI  \n• Exists optimal depth)
    F(Store Operations)
      H7(Store-to-store variance significant  \n• Compliance impacts Lift%)
    G(Frequency / Saturation)
      H8(Too frequent exposure → diminishing returns on Lift)
